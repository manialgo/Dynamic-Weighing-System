## Abstract

This project presents a fully functional **Dynamic Weighing System** capable of measuring the weight of objects while they are moving on a conveyor belt. The system achieves high precision of **±0.005 grams per kilogram** through careful mechanical design, vibration isolation, and software filtering.

---

## 1. Introduction & Objective

Traditional weighing systems require objects to be stationary. The main objective of this project was to build a system that can accurately weigh objects **in motion**, which has wide applications in industrial automation, logistics, and quality control.

---

## 2. System Design

### Mechanical Structure
- Two-stool support frame
- Triangular conveyor belt made with elastic rubber cloth
- PVC rollers with ball bearings
- Custom cones for belt tensioning
- Load cell platform mounted under the belt

### Electronics
- Arduino UNO / Nano
- HX711 Load Cell Amplifier
- 4 DC Motors with PWM control
- 16x2 LCD Display

---

## 3. Working Principle

Objects move on the conveyor → pass over the load cell zone → slight gravitational drop → weight is measured over several samples → average value displayed on LCD.

---

## 4. Implementation Highlights

- All mechanical work (cutting, welding, drilling) done by the team
- 75% of source code initially generated by AI and refined
- Over 15 days spent on calibration and error fixing
- Successfully sourced precise ball bearings after extensive search

---

## 5. Results

- **Accuracy Achieved**: ±0.005 g/kg
- Stable performance after multiple iterations
- Real-time weight display

---

## 6. Challenges & Solutions

- Belt vibration → Solved with proper tensioning and damping
- Inaccurate readings → Multiple calibration techniques + averaging algorithm
- Component sourcing → Dedicated one week for ball bearings

---

## 7. Conclusion & Future Scope

The project successfully demonstrates that dynamic weighing is possible with careful engineering. Future improvements can include:
- Higher speed conveyor
- Integration with IoT for data logging
- Machine learning for better accuracy
